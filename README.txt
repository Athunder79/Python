5 Aside leagues is a site designed to organize and track a football 5-aside league.

Users can enter an team and register for the league.

the site will generate a fixture list and allow user to inout scores for each game.

the site will also provide a league table and a standings table which is sortable.

It has been deployed on render.com and can be accessed here:  https://fiveasidetool.onrender.com
